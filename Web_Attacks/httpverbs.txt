#HTTP Verbs

Common methods:
GET, POST, HEAD, PUT, DELETE
#GET
ex: GET /page.php HTTP/1.1 
used to request a resource
can also be used to pass arguments to web server.
ex: GET /page.php?course=PTS HTTP/1.1
#POST
submit HTML form data. 
ex:
POST /login.php HTTP/1.1
HOST: www.example.site
username=john&password=mypass
#HEAD is similar to get. returns headers of response w/o response body.
#PUT is used to upload a file to the server.
Ex:
PUT /path/to/destination HTTP/1.1
Host: www.example.site
<PUT data>
#DELETE removes a file from the server. can lead to DoS and data loss
DELETE /path/to/destination HTTP/1.1
host: www.example.site
#OPTIONS queries server for enabled HTTP verbs
OPTIONS / HTTP/1.1
Host: www
#REST APIs specific web apps that rely on almost all HTTP verbs
- PUT for saving data & not files
